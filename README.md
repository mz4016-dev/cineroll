# CineRoll — Camera Roll Web App
A browser-based camera roll app built with vanilla HTML, CSS, and JavaScript — no frameworks.

## Features
- Live webcam photo and video capture
- Video recording with auto-generated thumbnails
- Photo filters via Canvas API (B&W, Vintage, Warm, Cool, Fade, Vivid, Drama)
- Custom scrollable film strip UI
- Slideshow mode with d-pad controls
- Lightbox viewer with zoom for photos and playback for videos
- Individual and bulk delete with restore bin
- Download any photo or video
- Date stamps on every frame
- localStorage persistence with image compression
